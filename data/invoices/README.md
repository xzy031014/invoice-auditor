# 发票图片目录

将待审核的发票图片文件放入此目录。

## 支持的格式

- `.jpg` / `.jpeg` - JPEG图片
- `.png` - PNG图片
- `.bmp` - 位图图片
- `.pdf` - PDF文件（需要额外处理）

## 文件命名建议

建议使用有意义的文件名，例如：
- `20240615_餐饮_张三.jpg`
- `20240610_差旅_李四.png`

程序会自动识别发票内容，文件名仅用于识别和追溯。

## 注意事项

1. 图片需要清晰可辨，建议分辨率不低于 1024x768
2. 发票关键信息（金额、日期、号码）需要清晰可见
3. 避免图片倾斜、模糊或光线不均匀
4. 建议使用原始扫描件或高清照片

## 示例发票结构

```
├── 20240615_restaurant.jpg      # 餐饮发票
├── 20240610_hotel.png           # 酒店发票
├── 20240605_transport.jpg       # 交通发票
└── 20240601_supplies.png        # 办公用品发票
```
